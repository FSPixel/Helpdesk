# Helpdesk IT Support

Dokumentasi pusat pengetahuan (*Knowledge Base*) untuk operasional dan pemecahan masalah IT Support. Proyek ini bertujuan untuk mendokumentasikan prosedur, solusi teknis, dan catatan operasional agar pengelolaan infrastruktur menjadi lebih terstruktur dan efisien.

## 🚀 Fitur Utama
* **Troubleshooting Guide**: Panduan langkah demi langkah untuk menangani masalah umum jaringan dan perangkat.
* **Operational Logs**: Pelacakan aktivitas IT harian dan pemeliharaan rutin.

## 🛠 Teknologi & Tools
* **Otomasi**: Google Apps Script (GAS)
* **Database**: Google Sheets
